Let's go for a sandwich

Baseline sandwich ⇒ a workload between 2 baseline runs

E(Workload) = E(measured) - (E(baseline-before)-E(baseline-after))/2

The ships that go first create waves that will alter the speed of those that arrive later. Any program will change the temperature of the processor so that the operating conditions for the next experiment will change. If we only use the one before, we might be missing part of the context. This is one of the innovations we have introduced in this paper, which allows to reduce inter-quartile range and thus have a more precise estimation of the energy consumed.
